Para que los datos se guarden en la nube, necesitás conectar Firebase. Solo hay que hacer esto una vez.

1
Crear el proyecto en Firebase
Entrá a console.firebase.google.com con tu cuenta de Google. Hacé clic en "Agregar proyecto", poné un nombre (ej: ordenes-juan), desactivá Google Analytics y creá el proyecto.
2
Crear la base de datos Firestore
En el menú izquierdo buscá "Firestore Database" → "Crear base de datos". Elegí "Comenzar en modo de prueba" y región us-east1.
3
Registrar la app web y pegar los datos
Engranaje ⚙️ → "Configuración del proyecto" → bajá a "Tus apps" → ícono </>. Poné un nombre y registrá.
const firebaseConfig = {
  apiKey: "← copiá",
  authDomain: "← copiá",
  projectId: "← copiá",
  appId: "← copiá"
};
Verificá los datos e intentá de nuevo.
